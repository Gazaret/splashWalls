# splashWalls
Beautiful photos from Unsplash in yours phone

## Getting Started

1. clone the repo   

```
git clone https://github.com/gazaret/splashWalls.git
```

2. install dependencies   
```
npm i
```

3. link dependencies  
```
react-native link
```

4. start in ios or android   

```
npm run ios
```
or
```
npm run android
```
